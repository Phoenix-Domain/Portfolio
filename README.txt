## Personal Portfolio Website

This project is a personal portfolio website designed to showcase my skills, projects, and experience. Built as a modern, responsive four-page site, it features vibrant colors, smooth animations, and a clean layout to provide visitors with an engaging overview of my professional background.

Features:
- Responsive design for optimal viewing on all devices
- Animated transitions and interactive elements
- Dedicated sections for About, Projects, and Contact
- Easily customizable content and styles

Technologies Used:
- HTML5, CSS3, and JavaScript
- AOS.js for interactivity and animations
- Font Awesome for icons

Credits:
- Inpiration and Starter Template by Chijioke Nwabasili
- Icons by Font Awesome (fontawesome.io)
- Responsive Tools (Chrome Dev Tools)

Feel free to explore, customize, and use this portfolio as inspiration for your own personal site!
