About Me

I’m a self-taught frontend developer passionate about building clean, responsive, and user-focused websites. My core skills include HTML, CSS, JavaScript (ES6+), along with experience in TailwindCSS, Bootstrap 5, and version control with Git/GitHub.

I’ve built and deployed practical, real-world projects such as booking dashboards, testimonial systems, portfolio websites, and interactive tools — all coded from scratch with clean and maintainable structure.

My approach is simple: solve real problems with functional, modern designs. I work with tools like Figma for design translation, focus on responsive layouts, and ensure smooth user experiences across devices.

Currently, I’m expanding into React and TypeScript development to build more dynamic, interactive web apps. I’m open to remote internships, collaborations, and client work.

Let’s bring your ideas to life with clean code and modern design.
